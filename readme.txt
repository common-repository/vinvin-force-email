== Description ==
Vinvin Force Email is usefull for testing and development purposes. It force all email send from website to being send to one email.
It change the subject of mail adding the original email. You can setup the email into WordPress general settings.
Im french [Wordpress developer](https://www.vinvin.dev/developpeur-wordpress-montreal/) based in Montreal.


== Installation ==
Just install the extension with classic method
and you set email into general settings
